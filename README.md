<br># FaceAuth - Facial Recognition System<br><br>## Overview<br><br>FaceAuth is a robust facial authentication system built with **Flask**, **OpenCV**, and deep learning techniques. It delivers seamless and secure access control for various applications, enabling user registration, facial recognition, and user management through an intuitive web interface.<br><br>---<br><br>## Features<br><br>- **User Registration:** Register new users by capturing multiple facial images.<br>- **Facial Recognition:** Authenticate users by comparing live camera feeds with registered data.<br>- **User Management:** View and delete registered users.<br>- **Model Training:** Automatically retrains when users are added or deleted.<br>- **Secure Sessions:** Uses Flask sessions for secure logins and dashboard access.<br>- **Error Handling:** Robust handling for camera, model, and file errors.<br><br>---<br><br>## Screenshots<br><br>| ![Homepage](![Screenshot 2025-06-08 180629](https://github.com/user-attachments/assets/1dad9fef-bea4-4531-b443-5f6f371bc3a0)<br>| ![Registration](![Screenshot 2025-06-08 180707](https://github.com/user-attachments/assets/cbd706f3-a904-4faf-b5c3-a7a2e2db8b45)<br>| ![Recognition](![Screenshot 2025-06-08 180725](https://github.com/user-attachments/assets/2bc273e5-0d54-42eb-9392-df1435a17bf5)<br>        <br>---<br><br>## Setup Instructions<br><br>1. **Clone the repository**<br>    ```bash<br>    git clone <your-repository-url><br>    cd facial<br>    ```<br>2. **Create and activate a virtual environment**<br>    - Windows:<br>        ```bash<br>        python -m venv venv<br>        .\venv\Scripts\activate<br>        ```<br>    - macOS/Linux:<br>        ```bash<br>        python -m venv venv<br>        source venv/bin/activate<br>        ```<br>3. **Install dependencies**<br>    ```bash<br>    pip install Flask opencv-python numpy<br>    ```<br>    > If you have issues with `opencv-python`, try `pip install opencv-python-headless`.<br>4. **Place pre-trained models**<br>    - Ensure `haarcascade_frontalface_default.xml` (from OpenCV) is available and accessible by your script.
